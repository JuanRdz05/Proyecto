Descripción del proyecto:

El proyecto consiste en el desarrollo de un sistema de gestión de citas para una clínica veterinaria, orientado a optimizar la administración de sus procesos. La plataforma permite gestionar usuarios, mascotas, citas médicas, servicios y registros clínicos, incorporando un control de acceso por roles que garantiza una organización segura y eficiente de la información.

Descripción de las carpetas:

- ROUTES: Contiene las rutas de la aplicación.
- CONTROLLERS: Contiene los controladores de la aplicación.
- MODELS: Contiene los modelos de la aplicación.
- CONFIG: Contiene la configuración de la aplicación.
- MIDDLEWARES: Contiene funciones de middleware para la aplicación.
